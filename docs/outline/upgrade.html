PHP 버전 호환성

그누보드는 PHP 5.2 ~ PHP 7 버전까지 최적화 되어있다.

PHP 8 버전에서도 대체로 동작하지만 일부 기능의 동작이 보장되지 않는다. 또한, 서드파티 테마, 스킨, 플러그인 등은 PHP 8 버전의 호환성이 더욱 떨어질 수 있다.
